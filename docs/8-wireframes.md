## 2. 메인 대시보드 레이아웃 (Main Dashboard)

### [전체 구조]
- **Sidebar**: 팀 목록 및 생성/가입 (모바일에서는 햄버거 메뉴 내 수납)
- **Header**: 알림, 개인 설정 (테마, 다국어)
- **Main**: 캘린더 영역
- **Interaction**: 날짜 클릭 시 해당 일자의 채팅/일정 상세 뷰 활성화

```text
+---+-------------------------------------------+
| T |  CalTalk Dashboard      [Lang] [Dark] [👤] | (Header)
+---+-------------------------------------------+
| T | < 2026.05 > [Month] [Week] [Day]  [+ Add] |
| 1 |-------------------------------------------|
+---| Sun | Mon | Tue | Wed | Thu | Fri | Sat |
| T |-------------------------------------------|
| 2 |     |     |     |     |     |     |     |
+---|  24 |  25 |  26 | [27]|  28 |  29 |  30 |
| + |     |     |     | [M] |     |     |     |
+---+-------------------------------------------+
(Sidebar)           (Calendar Main)
```
- `[M]`: 해당 날짜에 일정이 있음을 표시
- `[27]`: 현재 선택된 날짜 (클릭 시 하위 통합 뷰 오픈)

## 3. 반응형 통합 뷰 (Responsive Integrated View)

### 3.1 데스크톱 (Wide Screen: Split View)
- 화면 우측에 채팅 패널이 고정되어 캘린더와 동시 작업 가능.
- 캘린더 날짜를 클릭하면 우측 패널의 내용이 해당 날짜로 즉시 갱신.

```text
+---+----------------------------+-----------------------+
| T | CalTalk [Lang] [Dark] [👤] | [X] Chat: 2026.05.27  |
+---+----------------------------+-----------------------+
| T | < 2026.05 > [M][W][D] [+A] | [System] 2 Schedules  |
| 1 |----------------------------|-----------------------|
+---| Sun | Mon | Tue | Wed | Sat| [User A]: 미팅장소?   |
| T |----------------------------| [User B]: A-101호요   |
| 2 |     |     |     |     |    |                       |
+---|  24 |  25 |  26 | [27]|  30| [!] Change Request    |
| + |     |     |     | [M] |    |     14:00 -> 15:00    |
+---+----------------------------+     [Approve] [Reject]|
                                 |-----------------------|
                                 | [ Message...    ] [>] |
+--------------------------------+-----------------------+
(Sidebar)      (Calendar Area)          (Right Panel)
```

### 3.2 모바일/태블릿 (Small Screen: Toggle View)
- 공간 확보를 위해 채팅창은 기본적으로 숨김 처리.
- 상단 툴바의 **[Chat 🗨️]** 버튼을 통해 토글.

**[상태 A: 캘린더 화면]**
```text
+-------------------------------+
| [≡] CalTalk       [Dark] [👤] |
+-------------------------------+
| < 2026.05 >   [+Add] [Chat 🗨️] | <-- 채팅 토글 버튼
+-------------------------------+
| S | M | T | W | T | F | S |
+-------------------------------+
|   |   |   |[27]|  |   |   |
|   |   |   |[M]|   |   |   |
+-------------------------------+
| 27일 일정 (Brief)              |
| - 14:00 주간 회의             |
| - 16:30 코드 리뷰             |
+-------------------------------+
```

**[상태 B: 채팅 슬라이드 오버 (Slide-over)]**
- 캘린더 위로 채팅창이 슬라이드되어 나타나거나 전체 화면을 덮음.
```text
+-------------------------------+
| [← Back] Chat: 2026.05.27 [X] |
+-------------------------------+
| [System] 2 Schedules          |
|-------------------------------|
| [User A]: 미팅장소 어디인가요? |
| [User B]: A-101호로 오세요.    |
|                               |
| [!] 14:00 -> 15:00 변경 요청  |
|     [승인] [거절]              |
|                               |
|                               |
+-------------------------------+
| [ 메세지를 입력하세요... ] [>] |
+-------------------------------+
```

### 3.3 상태 전환 흐름 (State Transitions)
```text
[ Desktop (Wide Screen) ]
+------------------+      Click Date      +-------------------+
|    Calendar      |  ---------------->   | Calendar | Chat   |
|      Only        |  <----------------   |   (Split View)    |
+------------------+      Click [X]       +-------------------+

[ Mobile / Tablet (Small Screen) ]
+------------------+      Click [Chat]    +-------------------+
|    Calendar      |  ---------------->   |      Chat         |
| (Toggle Button)  |  <----------------   |   (Slide-over)    |
+------------------+    Click Back/[X]    +-------------------+
```

## 5. UI 가이드라인 (Responsive & Standard)

- **테마 모드 (Theme)**:
    - **Light**: White Background, Grey Borders, Blue Points (#007AFF).
    - **Dark**: Deep Navy Background (#1A1C1E), White Text, Cyan Points.
- **다국어 (Language)**:
    - KO/EN 토글 시 즉시 텍스트 리소스 변경.
- **반응형 전략 (Responsive Strategy)**:
    - **Breakpoint**: 1024px 이상 (Desktop Split View), 이하 (Mobile Toggle View).
    - **Sidebar**: 모바일에서는 Left Drawer(Hamburger)로 변환.
    - **Chat Panel**: 데스크톱은 고정(Fixed), 모바일은 오버레이(Overlay) 방식 적용.
- **애니메이션 (Motion)**:
    - 채팅창 등장 시 우측에서 좌측으로 Slide-in 효과 (Duration: 300ms, Easing: Ease-out).
